# Diploma App #
This is a full-stack project for theses assigment management for students and professors.

# Tech Stack #
Backend : Java, Spring Boot

Frontend : Javascript, Thymeleaf

Database : MySQL

Database Managment: phpMyAdmin

Testing : Mockito

# HOW TO RUN #
Open project on Intellij IDEA

XAMPP: Run As Administrator

On XAMPP select START: Apache, MySQL, Tomcat 

On Intellij -> MyWebApplication -> PLAY

Access app on localhost:8080
